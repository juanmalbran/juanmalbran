<h3 align="center">Nullsec · SIEM + Threat Intelligence</h3>

<p align="center">
  <strong>End-to-end threat detection cycle</strong>: an IOC loaded into MISP triggers a real SIEM alert after the controlled execution of ransomware (<strong>Bad Rabbit</strong>).<br/>
  I built the <strong>detection core</strong>: SIEM with ELK Stack (Elasticsearch · Kibana · Logstash), Fleet Server, <strong>ti_misp</strong> integration and <strong>4 KQL rules</strong> that detected the attack.
</p>

<p align="center">
  <strong>MTTD 5-6 min</strong> · 484 IOCs synced · 46 Sysmon events · validated detection of malicious PowerShell, C2, DNS and URLs.
</p>
